# 2) 작심삼일 루틴 챌린지 (총점 22)

## 핵심 가치
- “2주만 버티게 만드는 **체크인 루프**”
- “실패해도 재진입이 쉬운 UX”

## MVP 기능 (2주)
- 루틴 템플릿 + 커스텀 루틴 생성
  - 예: 운동/공부/절약/청소/독서
- 매일 체크인
  - 완료/미완료 + 한줄 메모(선택)
- 스트릭/배지
- 주간 리포트
  - 완료율, 가장 잘된 요일, 간단 코멘트(규칙 기반)
- “다시 시작” 버튼(중요)

## 화면/플로우
1. 루틴 선택/생성  
2. 오늘 체크인(원탭)  
3. 주간 리포트  
4. 루틴 설정(요일/목표)

## 데이터 모델(예시)
- `Routine(id, userId, title, daysOfWeek, goalPerDay, createdAt)`
- `Checkin(id, routineId, date, status, note)`

## 수익화
- 프리미엄
  - 루틴 개수 확장
  - 고급 통계(월/분기)
  - 템플릿 팩

## 2주 일정
- D1–3: 생성→오늘→리포트 핵심 루프
- D4–7: 스트릭/배지 + 빈상태/복귀 UX
- D8–10: 통계/리포트 확장 + 내보내기(선택)
- D11–14: QA/정책 문구/검수 대응

## 심사 리스크 최소화 포인트
- 과도한 푸시/강제 알림 금지(옵트인)
- “심리/의학적 치료” 표현 금지(습관 기록 도구로 한정)
